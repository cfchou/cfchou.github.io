Translate and Override Val/var/def

val/var/def are Scala’s language constructs that manifest immutibility amongst other features. Looking behind what scalac translates them to helps us to understand more about related semantics like object initialisation, override, etc..

Translate val/var/def to java

There are a few simple rules to translate val/var/def from scala to java.

For traits

* variables with mangled names to replace private variables; protected ones are NOT mangled
* getters and/or setters are generated
* initialisers are generated

For classes

* getters and/or setters are generated
